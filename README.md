## 📖 Project Overview

This project demonstrates the fundamentals of image processing using **OpenCV** and **Python**. It performs essential image manipulation techniques such as reading images, extracting image properties, converting color spaces, resizing, cropping, and saving processed images.

The project provides a strong foundation in computer vision and helps understand how digital images are represented and processed using OpenCV.

---

## 🚀 Features

- 📷 Read and display images
- 📊 Extract image properties
  - Shape
  - Size
  - Data Type
- ⚫ Convert image to Grayscale
- 🎨 Convert image to HSV Color Space
- 🌈 Convert image to LAB Color Space
- 📐 Resize images
- ✂️ Crop a Region of Interest (ROI)
- 💾 Save processed images
